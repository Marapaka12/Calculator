# 🧮 CalcFlex: Interactive Web Calculator with Dark Mode

A clean, responsive, and beautifully designed web calculator built using native web technologies. This project showcases structured DOM manipulation, crisp UI layout design, and dynamic runtime theming.

### 🛠️ Key Features
* **Core Math Engine:** Supports standard binary operations (addition, subtraction, multiplication, division), decimal precision sequencing, and immediate clear (`AC`) actions.
* **Dynamic Dark Mode Toggle:** Integrated a theme switcher that switches global color palettes smoothly by dynamically updating root CSS custom variables.
* **Robust Input Validation:** Handled standard input edge cases including duplicate decimal points, trailing operators, and division-by-zero errors.
* **Responsive Layout:** Designed utilizing a CSS Grid layout system to maintain perfect button alignment and structural scalability on any screen size.

### 🧰 Tech Stack
* **Markup:** HTML5 (Semantic Structure)
* **Styling:** CSS3 (Grid, Flexbox, Custom Variables)
* **Logic:** JavaScript (ES6+ Event Handling & DOM Manipulation)
